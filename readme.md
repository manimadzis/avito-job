## Вопросы, с которыми столкнулся
1. Каким типом данных представлены ID пользователя, услуги, заказа?

   uint64
2. Как задавать названия услуг, необхоимое для бух. отчета?

   Название услуги является опциональным параметром метода резервирования денег.
3. Как передавать JSON в GET запросы?

   Поскольку по ТЗ необходимо использовать JSON для приема запросов, то он передается в качестве path агрумента


## Инструкция по запуску
В корне проекта необходимо запустить
```
docker-compose up
```

## Swagger 
Swagger файл находится по следующему пути
```
api/v1.yaml
```
